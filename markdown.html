Markdown

Markdown är språket som används för att skapa inlägg på hemsidan. Här har du en snabb genom gång hur man formaterar ett sådant.

Titlar

{% highlight bash %} # H1 ## H2 ### H3 #### H4 ##### H5 ###### H6 {% endhighlight %}

Betoning

{% highlight bash %} italics: *asterisks* bold: **asterisks** italic & bold: _underscores_** genom strykning: ~~Scratch this.~~ {% endhighlight %}

Listor (numrerade och onumrerade)

{% highlight bash %} 1. artikel 2. artikel * indragen artikel * artikel * artikel 1 indragen artikel {% endhighlight %}

Länkar

{% highlight bash %} [google](https://www.google.com) {% endhighlight %} *Notera att addresser kommer automatiskt att omvandlar till länkar.
